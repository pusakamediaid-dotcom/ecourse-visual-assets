# ecourse-visual-assets

Visual assets (Islamic geometric SVG covers + infographics + onboarding deck)
for the E-Course Learning Workspace Notion draft.

## Layout

```
covers/                15 SVG covers (source, editable vector)
covers-png/            15 PNG covers (rasterized 1500px, used by Notion)
infographics/          2 SVG infographics (source)
infographics-png/      2 PNG infographics (used by Notion embed)
presentations/         Onboarding deck PPTX (7 slides)
```

## Usage

Raw URL format for Notion cover/embed:
```
https://raw.githubusercontent.com/pusakamediaid-dotcom/ecourse-visual-assets/main/<path>
```
