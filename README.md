# ecourse-visual-assets

Visual assets (Islamic geometric SVG covers + infographics) for the E-Course Learning Workspace Notion draft.

## Contents

```
covers/          15 SVG covers, one per Notion page (parent + 14 childs)
                 Unique composition, Islamic geometric pattern,
                 bright palette (emerald, teal, cream, gold, sage).

infographics/    2 SVG infographics for embedding in Notion pages:
                 - infographic-roadmap.svg  (Learning Journey diagram)
                 - infographic-quickguide.svg (8-step usage guide)
```

## Usage

Raw URL format for Notion cover/embed:
```
https://raw.githubusercontent.com/pusakamediaid-dotcom/ecourse-visual-assets/main/<path>
```
